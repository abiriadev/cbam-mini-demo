# Aipim CBAM reporter demo v2

## How to build and run

### Using Docker

```sh
$ docker compose up -d
```

or,

```sh
$ docker build . -t cbam-mini-demo:1
$ docker run --rm -p 13731:80 cbam-mini-demo:1
```

### Manual building

```sh
$ npm install
$ npm run build
$ npm run preview
```
